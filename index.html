Todo lo que necesitas en un reproductor moderno

Personalizable

Alto Rendimiento

Optimizado para cargar rápidamente y reproducir sin problemas incluso en conexiones lentas.

Instalación Rápida

1 Incluye las dependencias 

<!-- En el head de tu HTML -->
<script src="https://flixplayer.onrender.com/cdn/player.js"></script>

2 Agrega el contenedor HTML 

<!-- Donde quieras mostrar el reproductor -->
<div id="mi-reproductor" style="width: 100%; height: 400px;"></div>

3 Inicializa el reproductor 

<script>
    // Configuración básica del reproductor
    const player = new Player('mi-reproductor', {
        media: {
            title: "Mi Video",
            description: "Descripción del video",
            image: "imagen.jpg",
            file: "video.mp4"
        },
        player: {
            autostart: false,
            mute: false,
            volume: 80
        }
    });
</script>

Opciones de Configuración

Configuración de Media 

media: {
    title: "Título del video",       // String - Título que se mostrará
    description: "Descripción",     // String - Descripción del video
    image: "poster.jpg",            // String - URL de la imagen miniatura
    file: "video.mp4",              // String - URL del archivo de video
    downloadUrl: "video.mp4",       // String (opcional) - URL para descarga
    thumbnails: "thumbnails.vtt"    // String (opcional) - URL de miniaturas
}

Configuración del Reproductor 

player: {
    autostart: false,               // Boolean - Auto-reproducir al cargar
    mute: false,                    // Boolean - Silenciar al inicio
    volume: 80,                     // Number (0-100) - Volumen inicial
    displayDescription: true,       // Boolean - Mostrar descripción
    aboutText: "Powered by FlixPlayer", // String - Texto personalizado
    aboutLink: "",                  // String - Enlace para el texto
    logo: {                         // Object - Configuración de logo
        file: "logo.png",           // String - URL del logo
        link: "https://..."         // String - Enlace al hacer clic
    },
    captions: {                     // Object - Estilo de subtítulos
        color: "#FFF",              // String - Color del texto
        fontSize: 14,               // Number - Tamaño de fuente
        backgroundOpacity: 0,       // Number (0-1) - Opacidad fondo
        edgeStyle: "raised"         // String - Estilo del borde
    }
}

Características Avanzadas 

features: {
    seekButtons: true,              // Boolean - Botones de avance/retroceso
    progressSaving: true,           // Boolean - Guardar progreso de reproducción
    antiAdBlock: false,             // Boolean - Detectar AdBlock
    antiDownload: false             // Boolean - Prevenir descargas
}

Configuración de Monetización 

// Configuración básica de monetización
setMonetization({
    user: "https://tu-enlace.com"   // String - Tu enlace personalizado
});

// Configuración de tráfico de clics
setClickTraffic({
    enabled: true,                  // Boolean - Habilitar/deshabilitar
    interval: 300000,               // Number - Intervalo en milisegundos
    maxClicks: 10                   // Number - Máximo de clics permitidos
});

Métodos Públicos

play()

Inicia la reproducción del video.

player.play();
pause()

Pausa la reproducción del video.

player.pause();
loadMedia(file, image, title, description)

Carga un nuevo video en el reproductor.

player.loadMedia(
    "nuevo-video.mp4",   // String - URL del video
    "poster.jpg",        // String - URL de la miniatura
    "Título",            // String - Título del video
    "Descripción"        // String - Descripción
);
setDownloadUrl(url)

Establece una URL para descargar el video.

player.setDownloadUrl("video.mp4");
setMonetization(options)

Configura las opciones de monetización.

player.setMonetization({
    user: "https://tu-enlace.com"  // String - Enlace personalizado
});
destroy()

Elimina el reproductor y limpia los recursos.

player.destroy();

Ejemplos Prácticos

Reproductor Básico 

const basicPlayer = new Player('reproductor-basico', {
    media: {
        title: "Video Ejemplo",
        description: "Esto es un video de ejemplo",
        image: "poster.jpg",
        file: "video.mp4"
    },
    player: {
        autostart: false,
        mute: false,
        volume: 80
    }
});

Reproductor Protegido 

const protectedPlayer = new Player('reproductor-protegido', {
    media: {
        title: "Video Protegido",
        file: "video.mp4",
        image: "poster.jpg"
    },
    features: {
        antiAdBlock: true,   // Detectar AdBlock
        antiDownload: true   // Prevenir descargas
    }
});

Con Botón de Descarga 

const downloadPlayer = new Player('reproductor-descarga', {
    media: {
        title: "Video Descargable",
        file: "video.mp4",
        image: "poster.jpg",
        downloadUrl: "video.mp4"  // URL para descarga
    }
});

// También puedes agregarlo después:
downloadPlayer.setDownloadUrl("video.mp4");

Cambio Dinámico de Video 

const dynamicPlayer = new Player('reproductor-dinamico', {
    media: {
        title: "Primer Video",
        file: "video1.mp4",
        image: "poster1.jpg"
    }
});

// En algún evento (click, etc.)
document.getElementById('cambiar-video').addEventListener('click', () => {
    dynamicPlayer.loadMedia(
        "video2.mp4",
        "poster2.jpg",
        "Segundo Video",
        "Nueva descripción"
    );
});
