Reproductor de video con monetización inteligente y protección avanzada

Nuevas Características

Mejoras significativas en la versión 3.4

Monetización por Click

Anuncios se abren en nueva pestaña solo al hacer clic, sin redirección automática.

Protección Mejorada

Detección avanzada de AdBlock, IDM y entornos sandbox con alertas nativas.

Contador de Clics

Límite configurable de clics por sesión con temporizador de cooldown.

Instalación Rápida

1 Incluye las dependencias 

<!-- En el head de tu HTML -->
<script src="https://flix-player.onrender.com/cdn/player.js"></script>

2 Agrega el contenedor HTML 

<!-- Donde quieras mostrar el reproductor -->
<div id="mi-reproductor" style="width: 100%; height: 400px;"></div>

3 Inicializa el reproductor 

const player = new Player('mi-reproductor', {
    media: {
        title: "Mi Video",
        description: "Descripción del video",
        image: "imagen.jpg",
        file: "video.mp4"
    },
    monetization: {
        enabled: true,
        user: "https://tu-enlace-user.com",
        redirectMode: false // Anuncios se abren en nueva pestaña
    }
});

Configuración de Monetización

Opciones Básicas 

monetization: {
    enabled: true,          // Activar/desactivar monetización
    user: "https://...",
    redirectMode: false     // false = abre en nueva pestaña
}

Control de Clics 

monetization: {
    initialDelay: 10000,    // 10 segundos antes de activar clics
    cooldown: 15000,        // 15 segundos entre clics
    maxClicks: 3            // Máximo de clics por sesión
}

Sistema de Protección

Anti-AdBlock

Detecta extensiones bloqueadoras de anuncios y muestra alertas.

features: {
    antiAdBlock: true
}
Anti-Download

Bloquea gestores de descarga como IDM.

features: {
    antiDownload: true
}

Ejemplo Completo

const player = new Player('reproductor-premium', {
    media: {
        title: "Video Exclusivo",
        description: "Contenido premium protegido",
        image: "poster.jpg",
        file: "video.mp4",
        downloadUrl: null // Descargas desactivadas
    },
    player: {
        autostart: false,
        mute: false,
        aboutText: "Solo para miembros"
    },
    features: {
        antiAdBlock: true,
        antiDownload: true,
        progressSaving: true
    },
    monetization: {
        enabled: true,
        user: "https://afiliado.com",
        initialDelay: 15000,   // 15 segundos delay
        cooldown: 20000,      // 20 segundos entre clics
        maxClicks: 2,         // 2 clics máximo
        redirectMode: false    // Abre en nueva pestaña
    }
});
